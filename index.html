Video Game Average Score Classifier

For my capstone project I scraped Metacritic, the review site, for all-time top 100 games for 6 different consoles resulting in 112,345 reviews. I also Performed data cleaning, EDA, and preprocessing using popular libraries such as Matplotlib, Seaborn, Cleantext, and sklearn to create a dataset ready for modeling. I then built 2 models, Logistic Regression and Random Forests, along with Grid Search to produce interpretable results with the best accuracy score. The top 5 attributes correlating to a higher average user score were found and anaylzed.

Reddit post analysis using NLP

I utilized PRAW API to scrape Reddit “hot” posts from r/all over 5 days and collected 20,000 Reddit posts to use for Natural Language Processing and modeling. I then created K-Nearest Neighbor and Random Forests models using the sklearn library to identify a production model with the best accuracy score, and utilized Grid Search to find the optimal hyperparameters. I produced a confidence interval of 0.6556 ± 0.0120, 15% better than the baseline, and identified the top 5 features that increase the odds of a Reddit posts interaction.

Hackathon

This was a 6 hour hackathon project that I worked on with a small team of 3. We were tasked to utilize a dataset from Kaggle and perform EDA, fit and evaluate machine learning models. We chose to use the Kobe Bryant shot selection dataset and sought to find what characteristics of a shot lead to a higher probability of a shot being made. We used a logistic regression and random forests model. Our results were 61% accurate and showed that if Kobe was going for a dunk, he was very likely to make it.
